## What's New

### Student Entity
- Student entity with all required fields
- `StudentCode` auto-generated (`ST2026070001` format) by `StudentCodeGeneratorService`
- Basic info: name, gender, birthDate
- Contact info: phone, email
- School info: school, grade
- Tags: JSON Array (e.g. `["新生","重点"]`)
- Status enum: ACTIVE / PAUSED / GRADUATED / INACTIVE
- Soft delete (`deleted` flag) + Merge预留 (`mergedToStudentId`)
- `createdSource` tracking (WEB/IMPORT/API)
- Field-level change audit

### Parent-Student Relation
- `StudentParent` many-to-many association table
- One student ↔ multiple parents / one parent ↔ multiple students
- Relation type & isPrimary flag

### Student CRUD (7 endpoints)
| Method | Path | Description |
|--------|------|-------------|
| POST | /api/v1/students | Create student |
| GET | /api/v1/students | List (paginated, filterable) |
| GET | /api/v1/students/:id | Get by ID |
| PUT | /api/v1/students/:id | Update (field-level audit) |
| PATCH | /api/v1/students/:id/status | Change status |
| DELETE | /api/v1/students/:id | Soft delete |
| POST | /api/v1/students/import | Excel/CSV import |

### Parent-Student Endpoints (4 endpoints)
| Method | Path | Description |
|--------|------|-------------|
| POST | /api/v1/students/:id/parents | Link parent |
| DELETE | /api/v1/students/:id/parents/:parentId | Unlink parent |
| GET | /api/v1/students/:id/parents | Get student's parents |
| GET | /api/v1/students/parents/:parentId/students | Get parent's students |

### Import Service
- Reusable `ImportService` in `@utils/services/`
- Supports `.xlsx` and `.csv` formats
- Row-level validation with error collection
- Structured Import Report (total / success / failure / error details)
- `import_history` table reserved for future import tracking

### Student Audit
- `student_audit_log` table with full field tracking
- Creation audit: who, when, source
- Modification audit: field name, old value, new value, operator
- Status change audit with transition details
- Delete audit

### Business Rules
- `docs/BusinessRules/StudentRules.md` — Student domain business rules
- `docs/Review/Sprint-03-Checklist.md` — Gate review checklist

## Database Tables (3 new)
