# EduOS v0.0.2 — Sprint 2 Release

## Release Date
2026-07-06

## Summary
Identity Center (Authentication + RBAC) complete.

## What's New

### Authentication
- POST /api/v1/auth/login — BCrypt password hashing, JWT access token (2h)
- POST /api/v1/auth/logout — Clear refresh token, record login log
- POST /api/v1/auth/refresh — Rotate tokens, refresh token valid 7 days (DB-stored)
- GET /api/v1/auth/me — Current user profile

### Database Tables (6)
- `user` — Users table with BCrypt passwords
- `role` — Role definitions (SuperAdmin, Admin, Teacher, Parent)
- `permission` — Permission codes (12 initial permissions)
- `user_role` — User-role relations
- `role_permission` — Role-permission relations  
- `login_log` — Login/out/refresh audit trail

### Security
- JwtAuthGuard registered globally (all routes protected by default)
- @Public() decorator for public endpoints
- RolesGuard + @Roles() decorator for RBAC
- Refresh tokens stored in DB (supports admin kick-out)
- Password change invalidates all existing tokens

### Seed Data
- Admin user: `admin` / `admin123`
- 4 roles: SuperAdmin, Admin, Teacher, Parent
- 12 permissions across 6 modules

## Breaking Changes
- All API routes now require JWT auth by default (use @Public() to opt out)

## Technical Notes
- Requires MySQL 8.0 with EduOS database
- Run seed by integrating DatabaseModule → AppModule
