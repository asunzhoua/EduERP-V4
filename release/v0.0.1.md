# EduOS v0.0.1 — Sprint 1 Release

## Release Date
2026-07-06

## Summary
Foundation engineering complete. Zero business code.

## What's New

### Infrastructure
- NestJS 11 + TypeScript project initialized
- MySQL database connection configured (TypeORM)
- Unified config center (.env / .env.dev / .env.prod)
- 4-channel logging system (system, api, error, event)
- Unified API response format with global interceptor
- Global exception filter with error logging
- TypeScript path aliases configured

### Architecture
- EventBus framework with publish/subscribe
- 5 core events registered: LessonFinished, LeaveSubmitted, LeaveApproved, LessonFeedbackCreated, PointsGranted
- RBAC framework: RolesGuard + @Roles() decorator
- 4 roles scaffolded: Boss, Admin, Teacher, Parent

### Process
- .ai/ project memory directory established
- .architect/ architecture governance directory established
- Release notes system established

## Technical Notes

- Access token: 2h JWT, Refresh token: 7d (DB-stored)
- Password hashing: BCrypt
