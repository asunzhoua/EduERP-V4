## 备份策略

| 项目 | 配置 |
|------|------|
| 自动备份 | 每天凌晨 02:00 自动执行 |
| 备份内容 | 数据库、上传图片、系统配置、日志 |
| 保留周期 | 最近7天 / 最近30天 / 最近90天，超期自动删除 |
| 操作要求 | 先备份，后修改 |

## 恢复流程

```
停止系统 → 恢复数据库 → 恢复图片 → 恢复配置
→ 启动系统 → 检查数据 → 开放使用
```

## 恢复测试

每个月至少恢复一次备份，确认备份可用。

## 恢复原则

- 服务器异常、数据库异常、误操作 → 立即恢复
- **禁止人工修改数据库**

## 数据修改

所有数据修改只能通过后台操作，禁止直接操作数据库。
