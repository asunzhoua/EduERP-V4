## Service Patterns

### CRUD Pattern (Course, Class, Contract, Lesson)
```
create(dto, operatedBy) → Entity
findAll(query) → Entity[]
findByCode(code) → Entity
update(code, dto, operatedBy) → Entity
updateStatus(code, targetStatus, operatedBy) → Entity  [validates against VALID_TRANSITIONS]
remove(code, operatedBy) → void                        [soft delete or status change]
```

