## 验收标准

每个模块统一验收，检查以下项目全部通过方可验收：

### 功能清单

- 功能完成 — 与PRD定义一致
- 页面正常 — 按钮、输入框、列表、布局无错乱
- 权限正确 — 按RBAC设计验证每个角色
- 数据一致 — 多端数据实时一致
- 日志正常 — 所有修改记录审计日志
- 消息正常 — 通知准确，不重复不遗漏
- 无P0/P1 Bug

### Bug等级

| 等级 | 定义 | 处理要求 |
|------|------|----------|
| P0 | 系统无法使用（登录、签到、扣课、工资、数据库） | 立即修复，阻塞上线 |
| P1 | 影响正常使用（数据显示错误、页面异常、消息未发送） | 当天修复 |
| P2 | 体验问题（颜色、文字、排版） | 下一版本修复 |

### 最终上线条件

```
所有模块测试通过
所有P0 Bug清零
所有P1 Bug清零
数据库正常
服务器正常
消息正常
日志正常
权限正常
备份正常
恢复正常
```
